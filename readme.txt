== Installation & Activation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through Add New Plugin->Upload Plugin->Choose File choose dev-web-peronal-record.zip file.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After successful installation and activation, you will see an Admin Menu -> Dev Web Personal Record from this admin menu you can add new record, update existing record or delete record.
